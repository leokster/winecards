# Data Protection and GDPR Compliance

**App Name:** WineCards
**Bundle Identifier:** com.timrohner.winecards
**Last Updated:** November 24, 2024

## Overview

WineCards is fully compliant with GDPR (General Data Protection Regulation) and other data protection regulations because it does not collect, process, or store any personal data.

## Legal Basis

Under GDPR Article 4, "personal data" means any information relating to an identified or identifiable natural person. WineCards does not collect any such data.

## Data Processing Activities

### What Data is Processed:
1. **Starred Questions (Local Only)**
   - **Type:** User preferences (question IDs)
   - **Purpose:** Allow users to mark favorite questions
   - **Storage:** Local device only (iOS UserDefaults)
   - **Legal Basis:** User consent through app usage
   - **Retention:** Until app is uninstalled
   - **No Transfer:** Data never leaves the device

2. **App Content**
   - **Type:** Educational flashcard questions
   - **Purpose:** Provide learning material
   - **Storage:** Bundled with app, stored locally
   - **No Personal Data:** Questions contain no user information

### What Data is NOT Processed:
- ❌ No user accounts or profiles
- ❌ No email addresses or contact information
- ❌ No location data
- ❌ No device identifiers (IDFA, IDFV)
- ❌ No IP addresses
- ❌ No usage analytics or telemetry
- ❌ No cookies or tracking technologies
- ❌ No biometric data
- ❌ No photos or media access

## GDPR Rights

Since no personal data is collected, the following GDPR rights are not applicable:
- Right to access (Article 15)
- Right to rectification (Article 16)
- Right to erasure (Article 17)
- Right to restriction (Article 18)
- Right to data portability (Article 20)
- Right to object (Article 21)

However, users maintain complete control:
- **Delete all data:** Uninstall the app
- **Reset preferences:** Delete and reinstall

## Data Security Measures

### Technical Measures:
- All data stored using iOS secure storage (UserDefaults with app sandboxing)
- No network transmission of user data
- No external servers or databases
- iOS device encryption protects local data

### Organizational Measures:
- No access to user data (as none is collected)
- No data breach risk (no user data to breach)
- No third-party data processors

## International Data Transfers

**Not Applicable:** No data is transferred outside the user's device.

## Third-Party Services

WineCards does not use any third-party services that process personal data:
- No analytics providers
- No advertising networks
- No cloud storage services
- No social media integrations
- No payment processors

## External Content

The app loads images from:
- Domain: `elearning.academie-du-vin.ch`
- Purpose: Display wine label images in questions
- Data Sent: HTTP request for image URL only
- No Personal Data: No user identification in requests

## Data Protection Officer

Not required under GDPR Article 37, as:
1. No systematic monitoring of data subjects
2. No large-scale processing of special categories of data
3. No large-scale processing of personal data

## Supervisory Authority

Users in the EU have the right to lodge a complaint with their local data protection authority if they believe their rights have been violated.
